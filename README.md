# WeatherCheck
This application could be used to get weather information from any where in the world. 
Initialy it will ask permission to access geolocation. If user allows, it will display the weather status, temperature value in celcius unit, weather description (cloudy sky) and the user's city and country. Temperature can be viewed in Farenheit by clicking the temperature value. 
It will get the weather data from open weather map api.
Application is built using Javascript, HTML and CSS.
